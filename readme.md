# 易开发

> 易开发是一款帮助开发人员快速开发的工具，功能包括界面分析，页面信息等。目前软件尚未完成，更多功能完善中

### 支持设备

Android 5.0+

### 已支持功能

1. 应用基本信息查看包括：
 * versionCode/VersionName
 * TopActivity
 * 加固类型
 * Uid
 * 安装时间/更新时间
 * Data目录
2.  apk信息：
 * apk目录
 * MD5/Sha1
 * sharedPreference查看和编辑
 * 数据库查看
3.  布局分析
 * WidgetClass
 * ViewId
 * enable/clickable等状态
 * View文本内容
 * View位置
